商经法 - 第3讲 股东的出资

发表于2022-02-01|更新于2022-02-25|法考
|阅读量:

B站视频

1 出资方式

1.1 货币出资

  1. 无来源限制
  2. 存入公司账户

1.2 非货币出资

  1. 可估价 + 可转让
  2. 办理财产权转移手续
  3. 以不享有处分权的财产出资产生争议，遵循善意取得认定
  4. 自不动产实际交付时享有股东权利（非过户时）
  5. 土地使用权出资
    • 应未设定权利负担（不得已抵押）
    • 非划拨土地使用权（公益用地、非商业用地）
    • 处理：合理期限内解除负担、变更手续，否则认定未履行出资义务

1.3 股权出资

1.3.1 条件

  1. 合法持有并可转让
  2. 无权利瑕疵或权利负担（出资瑕疵、表决权委托、质押）
  3. 已办理股权转让手续
  4. 已价值评估

1.3.2 处理措施

合理期限内补正/评估，否则认定未全面履行出资义务

2 违反出资义务情形和处理

2.1 未按期缴足出资的处理

区别公司成立后进行增资，股东未按期履行出资义务的处理

2.1.1 对其他发起人的责任

承担违约责任（发起人协议或设立协议）

2.1.2 对公司的责任

  1. 请求依法履行出资义务
  2. 其他发起人承担连带责任

2.1.3 对公司债权人的责任

  1. 未出资本息范围内承担补充赔偿责任
  2. 其他发起人（不含公司成立后新加入的股东）与被告股东承担连带责任，承担责任后，可向被告股东追偿。
  3. 股东已经承担上述责任（未出资本息范围内），其他债权人提出相同请求，不予支持

2.1.4 未超出资期限股东的责任承担规则

  1. 请求未超出资期限股东承担补充赔偿责任，不予支持（股东享有期限权益）
  2. “出资加速到期”情形：
    • 公司已被受理破产
    • 公司作为被执行人，法院穷尽措施无财产可供执行，已具破产原因，但不申请破产的
    • 债务发生后，延长股东出资期限的

2.2 出资不实情形

指非货币财产的实际价额显著低于章程所定价额（恶意高估）

2.2.1 对其他发起人的责任

对其他发起人没有违约责任

2.2.2 对公司的责任

补足差额，其他股东连带责任

2.2.3 对债权人的责任

  1. 补充赔偿
  2. 其他发起人连带责任，事后向被告股东追偿
  3. 承担资产评估的机构，除非证明自己没有过错，否则承担不实金额范围内的赔偿责任

2.3 抽逃出资的情形

指未经法定程序，将出资抽回的行为。

情形：

  1. 虚增利润进行分配
  2. 虚构债权债务关系
  3. 利用关联关系
  4. 其他

其中，2、3有可能构成人格否认

第三人代垫资金协助发起人设立公司，第三人无需承担连带责任

2.3.1 对其他发起人的责任

对其他发起人没有违约责任

2.3.2 对公司的责任

  1. 返还出资本息
  2. 协助者连带（如董监高）

2.3.3 对公司债权人的责任

  1. 抽逃出资本息范围内补充赔偿
  2. 协助者连带

3 对违反出资义务股东的处理

3.1 限权

3.1.1 财产性权利的限制

根据公司章程或股东会决议做出合理限制

3.1.2 解除股东资格（特指有限公司）

股东大会决议

条件：

  1. 1分钱未出（？）
  2. 催告后仍未缴纳或返还

处理：

  1. 公司应及时办理减资程序或由其他股东或第三人缴纳相应的出资
  2. 公司债权人可请求相关当事人承担相应责任

3.1.3 另行募股（特指股份公司）

3.2 瑕疵股权的转让

未履行或未全面履行出资义务的，受让人知道或应当知道该情形，按以下规则处理：

  1. 受让人连带
  2. 公司无法偿债时，公司债权人可请求转让方承担补充赔偿责任，同时请求受让方承担连带请求
  3. 受让方承担责任后，可向转让方追偿。另有约定的除外

3.3 出资不适用诉讼时效抗辩

情形：

  1. 公司或其他股东请求其履行出资义务或返还出资
  2. 公司债权人在未过诉讼时效期间，当公司无法偿债时，债权人请求未履行出资义务的股东承担赔偿责任

股东以公司债权人对公司的债权已超过诉讼时效进行抗辩，经查属实的，法院予以支持。
