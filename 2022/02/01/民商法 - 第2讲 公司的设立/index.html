商经法 - 第2讲 公司的设立

发表于2022-02-01|更新于2022-02-25|法考
|阅读量:

B站视频

1 设立条件

1.1 发起人

  1. 合伙性质，应当签订发起人协议
  2. 共享利益，共担风险
  3. 有限公司1~50人，股份公司2~200人

1.2 公司章程

  1. 公司成立时，登记后才能生效
  2. 公司成立后，修改章程经股东大会表决即刻生效，无需等待登记
  3. 修改章程不登记，不得对抗善意第3人
  4. 公司超越章程而从事的经营活动，只要该行为没有违反法律，则该行为有效

1.3 注册资本

章程中记载，营业执照中载明。

1.4 公司名称和住所。

2 设立方式。

2.1 发起设立

  1. 发起人应当书面认购股份，并按章程缴纳出资
  2. 发起设立股份公司时，在发起人认购的股份缴足前，不得向他人募集股份

2.2 募集设立（重点）

2.2.1 注册资本

在公司登记机关中登记的实收股本总额。

2.2.2 设立程序

2.2.3 创立大会的职权

两审两人一通过

  1. 审议发起人公司筹办情况报告
  2. 通过公司章程（发起人制定，创立大会通过）
  3. 选举董事会和监事会
  4. 对公司的设立费用进行审核，对发起人用于抵作股款财产的作价进行审核
  5. 发出不可抗力或经营条件发生重大变化，可以作出不设立公司的决议

2.2.4 表决形式

创立大会必须经出席会议的认股人所持表决权过半数通过。

2.2.5 公司形式的变更

2.2.5.1 有限公司 → 股份公司
  1. 经代表2/3以上表决权的股东通过（全部表决权）
  2. 公司变更前的债权债务由变更后的公司继承
  3. 既可以采取发起设立，也可以采取募集设立
  4. 折合的实收股本总额不得高于公司净资产额
2.2.5.2 股份公司 → 有限公司

经出席会议股东所持表决权的2/3以上通过（出席）

3 设立责任。

合同责任，侵权责任、设立失败的责任。

3.1 设立中的合同责任。

  1. 发起人为设立公司，以自己名义对外签订合同：
  • 合同相对人可以请求该发起人承担责任。
  • 公司成立后，合同相对人可以请求公司承担责任。
  1. 发起人以设立中公司的名义对外签订合同：
  • 公司成立后，公司应当承担责任。
  • 公司成立后，有证据证明发起人利用设立中公司的名义为自己的利益与相对人签订合同，公司可以主张不承担合同责任，但相对人为善意的除外。

3.2 设立中的侵权责任。

发起人因履行公司设立职责，造成他人损害：

  • 公司成立后，公司承担责任。
  • 公司未成立，全体发起人承担连带责任。
  • 公司或者无过错的发起人承担责任后，可以向有过错的发起人追偿。

3.3 设立失败的责任。

公司因故未成立

  • 全体或者部分发起人对公司设立所产生的费用和债务承担连带清偿责任。
  • 由于发起人的过失导致公司利益受损，应当对公司承担赔偿责任。
  • 股份公司不能成立时，对于认股人已缴纳的股款，发起人负返还股款并加算银行同期存款利息的连带责任。
